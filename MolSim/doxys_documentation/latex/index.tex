\begin{DoxyItemize}
\item Refactoring\+:
\begin{DoxyItemize}
\item Encapsulated \doxylink{classParticle}{Particle} into \doxylink{classParticleContainer}{Particle\+Container} that can contain multiple particles in a Single Linked List
\item Single Linked List should be a good data structure for particle/molecule simulation tasks as it enables you to iterate through all particles very quickly and faster than in other data structures since the order in which you access the particles/molecules in a time step does not matter
\item Random access is not needed so you can do without the extra overhead other data structures need and can speed up the simulation process
\end{DoxyItemize}
\item I/O\+:
\begin{DoxyItemize}
\item Input immediately through command line from the beginning not while program runs
\item File name is needed to run the program, other values are optional and have default values if not specified
\item Possibility to easily add more options to the two already existing (delta\+\_\+time and end\+\_\+time) through existing structure
\item Output is generated by default in cmake-\/build-\/debug with both vtu-\/ and xyz-\/files for the simulation
\end{DoxyItemize}
